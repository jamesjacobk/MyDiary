MyDiary
Name of the database that it creates is "MyDiary"
It can be changed in the file 'index.php'.
Consequently, edits need to be made in other files.

Fonts used:
	"Open Sans" sourced from Google Fonts.

Uses jQuery 1.11.2.

There is a limit of 10000 characters per entry.

Default timezone is Asia/Kolkata. Set it to the desired timezone in process.php.

You can only add one entry per day.
